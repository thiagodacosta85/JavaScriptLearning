Estruturas If e Else

Operadores de comparação:

Igual (==) -> verifica se os valores comparados são iguais
Igual (===) -> verifica se os valores comparados são iguais e do mesmo tipo
Diferente (!==) -> verifica se os valores comparados são diferentes e de tipos diferentes
Menor ( < ) -> verifica se os valor é menor
Maior ( > ) -> verifica se os valor é maior
Menor igual ( <= ) -> verifica se os valor é menor
Maior igual ( >= ) -> verifica se os valor é maior
